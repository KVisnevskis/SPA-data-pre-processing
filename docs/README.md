# Documentation Index

Use this folder as the source of truth for pipeline behavior, config, and
output schema.

- `pipeline_specs.md`: Stage-by-stage processing specification.
- `data_schema.md`: Final HDF5 layout and column conventions.
- `manifest_reference.md`: Manifest structure and field definitions.
- `cli_reference.md`: Script entrypoints and practical usage examples.
- `github_issue_backlog.md`: Notes to convert into tracked issues.

Start with `../README.md` for setup and quickstart, then use the pages above
for detailed implementation-level reference.
